# Voice-to-Text

A Python application that transcribes audio files into text using Faster-Whisper.

## Features

- Fast speech-to-text transcription
- Outputs plain text files
- Built with Faster-Whisper

## Installation

```bash
git clone <repository-url>
cd voice-to-text
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
